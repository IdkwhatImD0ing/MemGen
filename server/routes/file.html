Our Project Overview

Welcome to our project! We have created an Express application with the following features:

  • Integration with Firebase for storing user data and text embeddings.
  • Ability to add text and associated embeddings to a user's collection in the Firestore database.
  • Query the data by finding the most relevant items based on a given user and text input.
  • Integration with OpenAI and Cohere to generate text based on a job description and previous experiences.
  • File upload and processing capabilities, including converting PDFs to text and storing as a Buffer.

This web application provides a powerful platform for text processing and content generation catering to job seekers and professionals.

API Endpoints

The following API endpoints are available:

POST /add
Adds text and associated embeddings. Requires the parameters userid and text.
POST /query
Queries the most relevant items based on the given user and text input. Requires the parameters userid and text.
POST /generate
Generates text based on a job description and previous experiences. Requires the parameters description and text.
POST /upload
Converts an uploaded PDF to text. Requires a PDF file uploaded with the field name pdf.
